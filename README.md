# [Odoo 开发规范](https://www.odoo.com/documentation/13.0/reference/guidelines.html)  

Odoo Guidelines介绍了Odoo编码标准。  
旨在提高Odoo 第三方模块的代码质量。高质量的代码可以提高代码的可读性，简化后期维护工作，帮助其他人员调试代码，降低代码复杂性并提高可靠性。这些准则应适用于每个新模块和所有新开发。
